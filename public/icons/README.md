# PWA icons

This MVP includes only `favicon.svg`. Before shipping a production PWA, drop the
following raster icons into this directory (Phase 8 will check them):

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512
- `icon-maskable-512.png` — 512×512, designed for the maskable safe-zone
- `apple-touch-icon.png` — 180×180

Generate them from `favicon.svg` with any icon-export tool (e.g. realfavicongenerator.net).
